### **PPIFuse (IEEE TCSVT 2025)**

***

This is the official implementation of the PPIFuse model proposed in the paper (PPIFuse: Physical Priors Injected Infrared and Visible Image Fusion) with pytorch.

- numpy 1.24.1
- tqdm 4.67.1
